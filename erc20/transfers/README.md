# ERC-20 Transfers

> Substreams for extracting ERC-20 events from EVM blockchains.

## Overview

This Substreams module extracts token-related events from Ethereum and EVM-compatible blockchains, including:

- **ERC-20 Events**: `Transfer`, `Approval`
